# Gereksinim Analizi

## Fonksiyonel Gereksinimler

- Sistem MySQL veritabanı için yedek alma işlemi gerçekleştirmelidir.
- Kullanıcı yedek dosyalarından geri yükleme yapabilmelidir.
- Sistem my.cnf yapılandırma dosyasını düzenleyebilmelidir.
- Kullanıcı etkileşimli bir menü aracılığıyla işlemleri seçebilmelidir.

## Fonksiyonel Olmayan Gereksinimler

- Sistem Windows işletim sistemi üzerinde çalışmalıdır.
- Python 3.10 ile uyumlu olmalıdır.
- Komut satırı tabanlı çalışmalıdır.
- Hata durumlarında kullanıcı bilgilendirilmelidir.

## Güvenlik Gereksinimleri

- Veritabanı kullanıcı bilgileri geçici olarak kullanılmalı ve kaydedilmemelidir.
- Yapılandırma dosyası düzenlenmeden önce otomatik yedek alınmalıdır.
