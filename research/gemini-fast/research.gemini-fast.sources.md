# Kaynak Listesi

Bu araştırma kapsamında yararlanılan temel teknik dokümantasyonlar ve endüstri standartları aşağıdadır:

1.  **PostgreSQL Official Documentation**: Backup and Restore Chapter (https://www.postgresql.org/docs/current/backup.html)
2.  **MySQL 8.0 Reference Manual**: Backup and Recovery (https://dev.mysql.com/doc/refman/8.0/en/backup-and-recovery.html)
3.  **Google Site Reliability Engineering (SRE) Book**: Data Integrity and Backup Strategies (https://sre.google/sre-book/data-integrity/)
4.  **OWASP (Open Web Application Security Project)**: Database Security Cheat Sheet (https://cheatsheetseries.owasp.org/cheatsheets/Database_Security_Cheat_Sheet.html)
5.  **Percona Database Performance Blog**: Best Practices for MySQL Backups.
